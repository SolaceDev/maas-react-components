<Subtitle>Component Description</Subtitle>

The Solace Pagination component provides navigation controls for large datasets split across multiple pages. It displays page numbers, navigation arrows, and result counts, allowing users to navigate through paginated content efficiently while maintaining awareness of their position within the dataset.

The component implements a pagination pattern with the purpose of breaking large datasets into manageable chunks while providing intuitive navigation controls. This pattern helps users browse through content efficiently while maintaining performance and usability.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(2) - Container spacing between pagination and content
- theme.spacing(0.5) - Item spacing between pagination items
- theme.components.MuiPagination.styleOverrides.root - Override pagination container styling
- theme.components.MuiPaginationItem.styleOverrides.root - Override pagination item styling
- theme.typography.body2 - Result count text styling

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and pagination behavior through pagination overrides.

**Layout Specifications:**

- Container spacing: theme.spacing(2) between pagination and content
- Item spacing: theme.spacing(0.5) between pagination items
- Page item size: 32px x 32px for consistent click targets
- Border radius: 4px for page items
- Result text: Typography body2 variant for result counts
- Responsive breakpoints: Simplified navigation on mobile devices
