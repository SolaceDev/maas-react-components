<Subtitle>Component Description</Subtitle>

The Solace Linear Progress component provides a visual indicator for ongoing processes or loading states with determinate and indeterminate progress display options. It's useful for showing the completion status of operations, implementing a progress indication pattern that helps users understand system activity and estimated completion times.

<Subtitle>Import Statement</Subtitle>

```tsx

```

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiLinearProgress-root` - Root linear progress container
- `.MuiLinearProgress-bar` - Progress bar element that animates
- `.MuiLinearProgress-bar1` - Primary progress bar for indeterminate variant
- `.MuiLinearProgress-bar2` - Secondary progress bar for indeterminate variant

**State Classes:**

- `.MuiLinearProgress-indeterminate` - Applied when variant is indeterminate
- `.MuiLinearProgress-determinate` - Applied when variant is determinate

**Color Classes:**

- `.default` - Default progress bar styling
- `.learning` - Learning-themed progress bar styling

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing() - Not directly used but can be applied through sx prop for margins
- theme.components.MuiLinearProgress.styleOverrides.root - Override default linear progress styling
- theme.components.MuiLinearProgress.styleOverrides.bar - Override progress bar styling

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, and customize the progress bar appearance through component overrides.

**Layout Specifications:**

- Height dimensions: xs: 4px, sm: 6px, md: 10px, lg: 14px
- Default height: md (10px)
- xs height: Used in modal dialog components
- sm height: Used for small progress indicators on side panels
- md height: Used for progress bars on main pages
- lg height: Used for prominent progress displays
