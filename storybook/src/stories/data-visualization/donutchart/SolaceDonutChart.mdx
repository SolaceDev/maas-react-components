<Subtitle>Component Description</Subtitle>

The Solace Donut Chart component is a versatile data visualization tool for displaying proportional or part-to-whole relationships. It presents data as segments of a circular chart, with an optional center that can contain an icon or be left empty. This component is particularly useful for showing percentage distributions, survey results, or any data that can be represented as parts of a whole.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.SolaceDonutChart-root` - Root container for the donut chart
- `.SolaceDonutChart-svg` - SVG element containing the chart visualization
- `.SolaceDonutChart-segment` - Individual data segments of the donut chart
- `.SolaceDonutChart-center` - Center area container for icon or content

**State Classes:**

- `.Mui-hovered` - Applied when chart segment is hovered
- `.Mui-active` - Applied when chart segment is active/selected
- `.SolaceDonutChart-empty` - Applied when no data is provided

**Segment Classes:**

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(1) - Segment stroke width spacing
- theme.spacing(2) - Tooltip padding spacing
- theme.typography.body2 - Tooltip text style
- theme.breakpoints.values.sm - Responsive chart sizing
- theme.components.SolaceDonutChart.styleOverrides.root - Root container overrides
- theme.components.SolaceDonutChart.styleOverrides.segment - Segment styling overrides

Available mixins for consistent styling:

- theme.mixins.chart.segment - Standard segment styling patterns
- theme.mixins.chart.tooltip - Consistent tooltip appearance

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and responsive behavior using breakpoint values.

**Layout Specifications:**

- Small chart dimensions: `120px x 120px`
- Medium chart dimensions: `200px x 200px`
- Large chart dimensions: `300px x 300px`
- Inner radius ratio: `0.6` (60% of outer radius)
- Segment stroke width: `2px`
- Tooltip padding: `8px 12px`
- Animation duration: `300ms`
