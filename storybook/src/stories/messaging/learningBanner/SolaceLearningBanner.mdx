<Subtitle>Component Description</Subtitle>

The Solace Learning Banner component provides a dismissible informational banner for displaying educational content, tips, or guidance to users. It's useful for highlighting learning resources, best practices, or helpful information without interrupting the main workflow, allowing users to access educational content when they need it.

The component implements an informational banner pattern with the purpose of providing contextual learning opportunities. This pattern helps users discover relevant educational content while maintaining focus on their primary tasks, with the option to dismiss the banner when no longer needed.

<Subtitle>When To Use It</Subtitle>

Use Solace Learning Banner when you need to provide educational or helpful information that enhances the user experience. This banner pattern is particularly effective for non-critical information that adds value.

Common use cases include:

- Displaying tips and best practices related to current user context
- Promoting learning resources or documentation links
- Showing feature announcements or new capabilities
- Providing contextual help or guidance for complex workflows
- Highlighting educational content without blocking user tasks
- Announcing training materials or tutorial availability

<Subtitle>When Not To Use It</Subtitle>

- For critical system messages or errors (use message box or alerts instead)
- When the information requires immediate user action (use confirmation dialogs instead)
- For temporary status updates (use toast notifications instead)
- When the banner would appear on every page load and become annoying
- For promotional content unrelated to user's current task (use dedicated promotion areas)
- When the content is too lengthy for a banner format (use dedicated help sections instead)

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `BannerContainer` - Root banner container with flex layout
- `TitleRow` - Container for title and close button when title is present
- `TitleSection` - Styled title text with subtitle1 typography

**State Classes:**

- Banner conditionally renders based on internal `open` state
- Layout changes between row and column based on presence of title

**Custom Classes:**

- `hasTitle` prop controls flex direction (column vs row)
- Custom styled components handle layout and spacing

**Theme Customization:**
The component uses Material-UI's theme system with custom Solace styling. Key theme paths:

- `theme.palette.ux.background.w20` - Default background color
- `theme.palette.ux.primary.text.wMain` - Text color
- `theme.typography.subtitle1.fontSize` - Title font size
- `theme.typography.fontWeightMedium` - Title font weight
- `theme.spacing(2)` - Container padding

**Layout Specifications:**

- Border radius: `4px` for rounded corners
- Container padding: `theme.spacing(2)` for consistent spacing
- Flex direction: `column` when title present, `row` when title absent
- Title typography: Uses `subtitle1` fontSize with medium font weight
- Close button: 24px icon size with icon variant styling
