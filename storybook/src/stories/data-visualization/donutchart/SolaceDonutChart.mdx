<Subtitle>Component Description</Subtitle>

The Solace Donut Chart component is a versatile data visualization tool for displaying proportional or part-to-whole relationships. It presents data as segments of a circular chart, with an optional center that can contain an icon or be left empty. This component is particularly useful for showing percentage distributions, survey results, or any data that can be represented as parts of a whole.

<Subtitle>When To Use It</Subtitle>

Use the Solace Donut Chart when you need to:

- Visualize part-to-whole relationships in your data
- Display percentage distributions
- Show the composition of a dataset
- Present survey results or statistical data
- Create a compact, visually appealing representation of proportional data
- Highlight a specific segment of data in relation to the whole

<Subtitle>When Not To Use It</Subtitle>

Avoid using the Solace Donut Chart in the following situations:

- When you have too many data points (more than 5-7), as it can become difficult to read
- For comparing absolute values across categories (use a bar chart instead)
- When the differences between segments are very small and hard to distinguish
- For time-series data or trends over time (use a line chart or area chart)
- When precise comparisons between values are required (consider a table or bar chart)
- If your audience is not familiar with interpreting donut charts

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.SolaceDonutChart-root` - Root container for the donut chart
- `.SolaceDonutChart-svg` - SVG element containing the chart
- `.SolaceDonutChart-segment` - Individual segments of the donut chart
- `.SolaceDonutChart-tooltip` - Tooltip displayed on hover (when enabled)
- `.SolaceDonutChart-icon` - Center icon container (when icon is provided)

**Theme Customization:**
The component uses Material-UI's theme system with custom Solace styling. Key theme paths:

- `theme.palette.ux.primary` - Default color for chart segments if not specified in data
- `theme.palette.ux.secondary` - Used for tooltip background
- `theme.typography.body2` - Applied to tooltip text

**Layout Specifications:**

- Chart size: Determined by the `size` prop (sm, md, lg)
- Segment stroke: 1px for clear segment separation
- Tooltip padding: Consistent padding for readability
- Center icon placement: Centered both horizontally and vertically
- Responsive design: Adjusts to container width while maintaining aspect ratio
