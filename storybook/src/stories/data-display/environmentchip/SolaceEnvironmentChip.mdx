<Subtitle>Component Description</Subtitle>

The SolaceEnvironmentChip component provides a visual indicator for displaying environment-specific information with customizable styling and colors, useful for showing environment states, deployment stages, or system status indicators that allow users to quickly identify and understand the current environment context or operational state. This environment badge pattern provides clear visual distinction between different environments or system states, helping users maintain situational awareness about their current working context while providing consistent visual styling across different environment types.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing([NUMBER]) - Padding, margin, border radius, and gaps (e.g., theme.spacing(3) for height)
- theme.typography.body1 - Font family, size, and weight for text content
- theme.breakpoints.values.[SIZE] - Responsive behavior breakpoints
- theme.components.MuiChip.styleOverrides.root - Custom styling for chip container
- theme.components.MuiChip.styleOverrides.label - Custom styling for chip text
- theme.components.MuiChip.styleOverrides.icon - Custom styling for leading icon
- theme.components.MuiChip.styleOverrides.deleteIcon - Custom styling for delete icon

Available mixins for consistent styling:

- theme.mixins.toolbar - Consistent toolbar styling if applicable in chip groups

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and responsive behavior using breakpoint values.

**Layout Specifications:**

- Height: `theme.spacing(3)` (24px typically)
- Border radius: `theme.spacing(0.5)` (4px typically)
- Border width: `1px` solid
- Default max width: `200px`
- Icon size: `16x16` pixels (for both leading icon and close icon)
- Column gap: `theme.spacing(0.5)` (4px typically) between icon and text
- Padding variations:
  - Icon only: `theme.spacing(0, 0.5)` (0px vertical, 4px horizontal)
  - With delete: `theme.spacing(0, 0.5)` (0px vertical, 4px horizontal)
  - Standard: `theme.spacing(0, 1, 0, 0.5)` (0px top, 8px right, 0px bottom, 4px left)
- White background special case: Adds border using `theme.palette.ux.secondary.w100`
- Text overflow: Ellipsis with `white-space: nowrap`
- Transition timeout: `200ms`
