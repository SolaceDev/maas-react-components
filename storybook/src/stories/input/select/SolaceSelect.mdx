<Subtitle>Component Description</Subtitle>

The Solace Select component provides a customizable dropdown selection field with support for various display formats and states. It's useful for creating accessible and feature-rich selection interfaces, allowing users to choose from a predefined list of options with support for icons, subtexts, and custom styling.

The component implements a flexible selection pattern with the purpose of handling both simple and complex option displays. This pattern helps users make selections efficiently while maintaining accessibility and providing rich visual feedback through various states and customization options.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiSelect-root` - Root select container
- `.MuiSelect-select` - Select input element
- `.MuiSelect-icon` - Dropdown icon container
- `.MuiMenuItem-root` - Menu item container

**State Classes:**

- `.Mui-focused` - Applied when the select has focus
- `.Mui-disabled` - Applied when the select is disabled
- `.Mui-error` - Applied when hasErrors is true
- `.readOnlySelect` - Applied when readOnly is true
- `.emptySelect` - Applied when no value is selected

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.components.MuiSelect.styleOverrides.root - Customize the root select container
- theme.components.MuiSelect.styleOverrides.select - Customize the select input element
- theme.components.MuiSelect.styleOverrides.icon - Customize the dropdown icon
- theme.components.MuiMenuItem.styleOverrides.root - Customize menu item styling
- theme.components.MuiMenu.styleOverrides.paper - Customize the dropdown menu paper
- theme.components.MuiMenu.styleOverrides.list - Customize the dropdown menu list

To customize layout and structure, override the component styleOverrides in your theme configuration.

**Layout Specifications:**

- Select height: 32px (theme.spacing(4))
- Select padding: 8px 32px 8px 12px (theme.spacing(1, 4, 1, 1.5))
- Menu max height: 300px (customizable through maxHeight prop)
- Menu width: Matches select width by default
- Menu item height: 36px
- Menu item padding: 6px 16px
- Border radius: 4px (theme.shape.borderRadius)
- Border width: 1px
- Font size: 14px
- Line height: 1.4375em
- Transition: 150ms cubic-bezier(0.4, 0, 0.2, 1) for background-color, border-color and box-shadow
