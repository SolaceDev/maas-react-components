<Subtitle>Component Description</Subtitle>

The Solace Categorized Search component provides a powerful search interface that combines text-based searching with category filtering, useful for narrowing down search results within specific categories and allowing users to efficiently find content across different sections or types of data. The component implements a combined search pattern with the purpose of enhancing search precision through categorization, helping users quickly locate relevant information by combining free-text search with predefined category filters while maintaining a clean and intuitive interface.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiTextField-root` - Root search input container
- `.MuiToggleButtonGroup-root` - Category buttons container
- `.MuiToggleButton-root` - Individual category button
- `.MuiFormHelperText-root` - Helper text container

**State Classes:**

- `.Mui-focused` - Applied when the search input has focus
- `.Mui-selected` - Applied to the currently selected category button
- `.Mui-disabled` - Applied when the component is disabled
- `.Mui-error` - Applied when the component has an error state

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(1) - Padding between category buttons (4px)
- theme.typography.body1 - Text style for search input
- theme.typography.button - Text style for category buttons
- theme.components.MuiTextField.styleOverrides.root - Search input container customization
- theme.components.MuiToggleButton.styleOverrides.root - Category button customization

Available mixins for consistent styling:

- theme.mixins.toolbar - Used for maintaining consistent height with toolbar components

**Layout Specifications:**

- Search input padding: theme.spacing(2) vertical, theme.spacing(1.5) horizontal
- Category button spacing: theme.spacing(1) (4px) between buttons
- Button height: 40px (matches MUI default input height)
- Transition timeout: 200ms for hover/focus effects
