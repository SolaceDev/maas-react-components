<Subtitle>Component Description</Subtitle>

The Solace Toggle Button Group component provides a set of mutually exclusive toggle buttons for selection. It's useful for filtering, categorization, or switching between different view states, allowing users to select one option from a group of related choices.

The component implements an exclusive selection pattern with the purpose of providing clear visual feedback for the active state. This pattern helps users understand their current selection while still providing easy access to switch between options.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiToggleButtonGroup-root` - Root toggle button group container
- `.MuiToggleButtonGroup-grouped` - Individual toggle button elements
- `.MuiToggleButton-root` - Individual button styling

**State Classes:**

- `.Mui-selected` - Applied when a toggle button is selected/active
- `.Mui-disabled` - Applied when the button group or individual button is disabled
- `.Mui-focusVisible` - Applied when a button receives keyboard focus

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.components.MuiToggleButtonGroup.styleOverrides.root - Customize the root toggle button group container
- theme.components.MuiToggleButtonGroup.styleOverrides.grouped - Customize individual toggle button elements
- theme.components.MuiToggleButton.styleOverrides.root - Customize individual button styling

To customize layout and structure, override the component styleOverrides in your theme configuration.

**Layout Specifications:**

- Button height: 32px (theme.spacing(4))
- Button padding: 0 16px (theme.spacing(0, 2))
- Border style: 1px solid with theme colors
- Font weight: Regular (400) for unselected, Medium (500) for selected
- Text transform: none (preserves original casing)
- Border radius: 4px (theme.shape.borderRadius)
- Transition: 150ms cubic-bezier(0.4, 0, 0.2, 1) for background-color and box-shadow
