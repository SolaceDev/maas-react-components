<Subtitle>Component Description</Subtitle>

The Solace Stepper component provides a step-by-step navigation interface for multi-step processes and forms. It displays a vertical list of steps with progress indicators, allowing users to navigate through sequential workflows while maintaining awareness of their current position and completion status.

The component implements a stepper pattern with the purpose of breaking complex processes into manageable steps. This pattern helps users complete lengthy forms or workflows by providing clear progression indicators and navigation controls.

<Subtitle>When To Use It</Subtitle>

Use Solace Stepper when you need to guide users through multi-step processes or workflows. This stepper pattern is essential for complex sequential tasks.

Common use cases include:

- Multi-step form wizards and configuration processes
- Onboarding flows for new users or features
- Setup and installation processes
- Data entry workflows with validation at each step
- Complex business processes requiring sequential completion
- Guided tutorials or walkthroughs

<Subtitle>When Not To Use It</Subtitle>

- For simple forms that can be completed on a single page
- When steps are not sequential or can be completed in any order - use tabs instead
- For read-only content or information display - use different navigation patterns
- When there are too many steps (more than 6-8) - consider grouping or simplifying
- For mobile experiences where vertical space is limited
- When users frequently need to jump between non-adjacent steps

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiStepper-root` - Root stepper container
- `.MuiStep-root` - Individual step container
- `.MuiStepLabel-root` - Step label container
- `.MuiStepIcon-root` - Step icon container
- `.MuiStepContent-root` - Step content area
- `.MuiStepConnector-root` - Connector line between steps

**State Classes:**

- `.Mui-active` - Applied to the currently active step
- `.Mui-completed` - Applied to completed steps
- `.Mui-disabled` - Applied to disabled steps
- `.Mui-error` - Applied to steps with errors

**Custom Classes:**

- Custom step icon styling with error and completed states
- Footer navigation with previous/next button styling
- Step content area with proper padding and spacing
- Progress indicator styling for visual feedback

**Theme Customization:**
The component uses Material-UI's theme system with custom Solace styling. Key theme paths:

- `theme.palette.ux.primary.wMain` - Active step styling
- `theme.palette.ux.error.wMain` - Error state styling
- `theme.palette.ux.success.wMain` - Completed step styling
- `theme.palette.ux.secondary.w40` - Inactive step styling

**Layout Specifications:**

- Stepper width: Configurable sidebar width for step navigation
- Content area: Flexible width for step content
- Footer height: Fixed height for navigation buttons
- Step spacing: `theme.spacing(2)` between steps
- Icon size: `24px x 24px` for step icons
- Button spacing: `theme.spacing(1)` between footer buttons
