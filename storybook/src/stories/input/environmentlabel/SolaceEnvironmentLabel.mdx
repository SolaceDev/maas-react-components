<Subtitle>Component Description</Subtitle>

The Solace Environment Label component provides a visual identifier for environments with colored icons and text labels. It's useful for displaying environment information in a compact, easily recognizable format, allowing users to quickly identify different environments by their distinctive colors and icons.

The component implements a horizontal layout pattern with the purpose of showing environment context at a glance. This pattern helps users distinguish between different environments while maintaining visual consistency across the application.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- Custom styled components (no standard MUI classes)
- `Container` - Root label container with inline-flex layout
- `ColoredIcon` - Icon container with colored background
- `Label` - Text label with overflow handling

**Variant Classes:**

- `variant="standard"` - Standard label styling with body1 typography
- `variant="title"` - Title styling with h1 typography for prominence

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.typography.body1 - Standard variant typography
- theme.typography.h1 - Title variant typography
- theme.spacing(3) - Component height and icon minimum width
- theme.spacing(2) - Gap between icon and label
- theme.spacing(0.5) - Icon border radius

**Layout Specifications:**

- Container height: `24px` (theme.spacing(3))
- Icon minimum width: `24px` (theme.spacing(3))
- Gap between elements: `16px` (theme.spacing(2))
- Icon border radius: `4px` (theme.spacing(0.5))
- Text overflow: `ellipsis` with `white-space: nowrap`
- Icon border: `1px solid` (transparent unless white background)
- Container layout: `inline-flex` with `align-items: center`
- Label max-width: `calc(100% - 40px)` to accommodate icon and spacing
