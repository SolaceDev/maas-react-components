<Subtitle>Component Description</Subtitle>

The Solace Tabs component provides horizontal tab navigation for organizing content into multiple sections. It's useful for creating tabbed interfaces with text-based tab labels, implementing a tab navigation pattern that allows users to switch between different content views while maintaining context and providing clear section organization.

<Subtitle>Import Statement</Subtitle>


<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiTabs-root` - Root tabs container
- `.MuiTabs-flexContainer` - Flex container for tab items
- `.MuiTabs-indicator` - Active tab indicator line
- `.MuiTab-root` - Individual tab button
- `.MuiTab-labelIcon` - Tab with icon styling
- `.MuiTabs-scrollButtons` - Scroll navigation buttons for scrollable variant

**State Classes:**

- `.Mui-selected` - Applied to the currently active tab
- `.Mui-disabled` - Applied to disabled tabs
- `.Mui-focusVisible` - Applied when tab has keyboard focus

**Variant Classes:**

- `.MuiTabs-standard` - Standard spacing variant
- `.MuiTabs-fullWidth` - Full width variant where tabs fill container
- `.MuiTabs-scrollable` - Scrollable variant for overflow content

**Custom Classes:**

- Custom active indicator styling with Solace brand colors
- Notification counter integration for tab labels
- Custom hover states and focus indicators
- Typography scaling based on size prop

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(1, 2) - Tab padding for comfortable click targets
- theme.components.MuiTabs.styleOverrides.root - Override tabs container styling
- theme.components.MuiTab.styleOverrides.root - Override individual tab styling
- theme.typography.button - Tab text typography styling
- theme.shape.borderRadius - Border radius for focus indicators

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and tab behavior through tabs overrides.

**Layout Specifications:**

- Tab height: 48px for standard size, 56px for medium size
- Indicator height: 2px positioned at bottom of active tab
- Tab padding: theme.spacing(1, 2) for comfortable click targets
- Font sizes: sm (14px) default, md (16px) for larger variant
- Border radius: 4px for focus indicators
- Scroll button size: 40px x 40px for scrollable variant
