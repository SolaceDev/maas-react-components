<Subtitle>Component Description</Subtitle>

The Solace File Uploader component provides a drag-and-drop interface for file uploads with progress tracking and validation, useful for applications that require file handling capabilities and allowing users to easily upload single or multiple files while providing visual feedback on the upload status and any validation errors. The component implements a comprehensive file upload pattern with the purpose of making file selection and upload intuitive and user-friendly, helping users understand file requirements, track upload progress, and handle any validation or upload errors effectively.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.dropzone` - Root container for the drag-and-drop area
- `.dropzone--active` - Applied when files are being dragged over
- `.dropzone--disabled` - Applied when uploader is disabled
- `.file-list` - Container for uploaded file items
- `.file-item` - Individual file item in the list

**State Classes:**

- `.file-item--uploading` - Applied during file upload
- `.file-item--error` - Applied when file has validation errors
- `.file-item--success` - Applied when file upload is complete
- `.progress-bar` - Upload progress indicator container
- `.progress-bar--determinate` - Progress bar with known progress
