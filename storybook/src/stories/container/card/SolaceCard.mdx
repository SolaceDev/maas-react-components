<Subtitle>Component Description</Subtitle>

The SolaceCard component provides a customizable card container for displaying learning resources, content sections, and interactive elements with optional headers, content areas, and action sections. This flexible card layout pattern helps organize complex interfaces into visually distinct content containers that can serve both as information displays and navigation entry points, allowing users to scan and interact with grouped content efficiently while maintaining clean visual hierarchy and consistent spacing.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiCard-root` - Root card container
- `.MuiCardContent-root` - Content section of the card
- `.MuiCardActions-root` - Actions section at the bottom of the card

**State Classes:**

- `.solaceCard` - Applied to all SolaceCard instances
- `.readOnly` - Applied when card is in read-only mode
- `.MuiCardContent-root:last-child` - Removes bottom padding when content is the last child

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing([NUMBER]) - Padding and margin (e.g., theme.spacing(2) for 16px)
- theme.typography.[VARIANT] - Text styles for card content
- theme.breakpoints.values.[SIZE] - Responsive behavior breakpoints
- theme.components.MuiCard.styleOverrides.root - Custom styling for card container
- theme.components.MuiCardContent.styleOverrides.root - Custom styling for content area
- theme.components.MuiCardActions.styleOverrides.root - Custom styling for actions area

Available mixins for consistent styling:

- theme.mixins.toolbar - Consistent toolbar styling if applicable in card actions

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and responsive behavior using breakpoint values.

**Layout Specifications:**

- Default padding: `theme.spacing(2)` (16px)
- Minimum width: `250px` (default)
- Content padding: `theme.spacing(0)` (removes MUI default padding)
- Actions padding: `theme.spacing(0)` (removes MUI default padding)
- Content bottom padding: `theme.spacing(0)` when content is last child
- Role: `article` for accessibility
