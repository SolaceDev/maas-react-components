<Subtitle>Component Description</Subtitle>

The Solace Page Header component provides a standardized header section for pages that includes title, breadcrumb navigation, and action buttons. It's useful for creating consistent page layouts with proper navigation context, implementing a page header pattern that establishes clear page hierarchy and provides quick access to primary page actions.

**Main Component Classes:**

- Custom `HeaderContainer` - Main header container with border-top styling
- Custom `TopRowContainer` - First row containing breadcrumbs and environment
- Custom `SecondRowContainer` - Second row with title, subtitle, and actions
- Custom `ThirdRowContainer` - Third row for tabs and icon tabs
- Custom `BreadcrumbContainer` - Container for breadcrumb navigation
- Custom `TitleContainer` - Container for title and subtitle
- Custom `ActionsContainer` - Container for action buttons and menu

**Layout Classes:**

- Responsive flexbox layout with proper spacing

- Action button grouping with consistent spacing
- Tab integration with proper alignment

**Custom Classes:**

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(2, 3) - Header padding for consistent margins
- theme.spacing(1) - Row spacing between header rows and action buttons
- theme.typography.h1 - Main title typography styling
- theme.typography.subtitle1 - Subtitle typography styling
- theme.breakpoints.down('md') - Responsive behavior breakpoints

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and responsive behavior through breakpoint utilities.

**Layout Specifications:**

- Header padding: theme.spacing(2, 3) for consistent margins
- Row spacing: theme.spacing(1) between header rows
- Border-top: 2px solid with environment-matched color
- Action spacing: theme.spacing(1) between action buttons
- Responsive breakpoints: Proper stacking on mobile devices
- Title typography: H1 for main title, subtitle styling for secondary text
