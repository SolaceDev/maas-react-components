<Subtitle>Component Description</Subtitle>

The Solace Truncatable Link component provides clickable text elements that automatically truncate with ellipsis when content exceeds the available space. It includes tooltip functionality to display the full text on hover, ensuring users can always access complete information while maintaining clean layouts in constrained spaces.

The component implements a truncatable link pattern with the purpose of displaying long text content in limited space while preserving functionality. This pattern helps maintain visual consistency in layouts while ensuring important information remains accessible through progressive disclosure.

<Subtitle>When To Use It</Subtitle>

Use Solace Truncatable Link when you need to display potentially long link text in constrained spaces. This truncation pattern is essential for maintaining clean layouts with variable content lengths.

Common use cases include:

- Table cells with long URLs or file names
- Navigation items with variable-length labels
- List items with lengthy titles or descriptions
- Inline links within content blocks with space limitations
- Card components with dynamic text content
- Breadcrumb navigation with long page names

<Subtitle>When Not To Use It</Subtitle>

- When the full text is critical and must always be visible
- For very short text that doesn't need truncation
- When horizontal scrolling is preferred over truncation
- For text that users need to copy exactly - provide full text alternative
- In contexts where tooltips may not be accessible (touch-only interfaces)
- When text length is predictable and fits available space

Structural and layout customization through component overrides and mixins:

- theme.components.MuiButton.styleOverrides.textPrimary - Override link button styling

- theme.typography.body2 - Link text typography styling
- theme.spacing() - Internal padding and margins

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and tooltip behavior through tooltip overrides.

**Layout Specifications:**

- Container: max-width: 100% to respect parent constraints
- Text truncation: Ellipsis appears when content overflows
- Tooltip positioning: bottom-end placement by default
- Button padding: Inherited from SolaceButton link variant
- Focus indicators: 1px solid outline with 1px offset
- Hover states: Background color changes on interactive elements

**Accessibility Features:**

- Tooltip provides full text for screen readers
- Proper focus management for keyboard navigation
- ARIA labels for link context and behavior
- Semantic link structure for assistive technologies
