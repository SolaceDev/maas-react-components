<Subtitle>Component Description</Subtitle>

The Solace Toggle component is a customizable switch input that allows users to toggle between two states. It's designed for use in forms and other UI elements where a binary choice is required. The component extends the functionality of a standard switch input, adding features such as labels, error states, helper text, and customizable styling.

This component is particularly useful for creating accessible and visually consistent toggle inputs that fit within the Solace design system. It supports both controlled and uncontrolled usage, making it flexible for various application needs.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiSwitch-root` - Root switch container
- `.MuiSwitch-switchBase` - The base of the switch
- `.MuiSwitch-thumb` - The circular thumb of the switch
- `.MuiSwitch-track` - The track that the thumb slides on

**State Classes:**

- `.Mui-checked` - Applied when the switch is in the on state
- `.Mui-disabled` - Applied when the switch is disabled
- `.Mui-focusVisible` - Applied when the switch has keyboard focus

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.components.MuiSwitch.styleOverrides.root - Customize the root switch container
- theme.components.MuiSwitch.styleOverrides.switchBase - Customize the base of the switch
- theme.components.MuiSwitch.styleOverrides.thumb - Customize the circular thumb of the switch
- theme.components.MuiSwitch.styleOverrides.track - Customize the track that the thumb slides on

To customize layout and structure, override the component styleOverrides in your theme configuration.

**Layout Specifications:**

- Switch Size: small (Material-UI default small size)
- Switch Width: 34px
- Switch Height: 18px
- Thumb Diameter: 14px
- Label Position: To the right of the switch
- Helper Text: Below the switch and label
- Spacing between switch and label: 8px (theme.spacing(1))
