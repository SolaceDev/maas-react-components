<Subtitle>Component Description</Subtitle>

The Solace Label component provides a customizable label for form elements, useful for creating accessible and visually consistent form labels that allow users to clearly identify and understand the purpose of associated input fields. The component implements a flexible labeling system with the purpose of enhancing form usability and accessibility, helping users navigate forms more efficiently while maintaining a clean and organized layout.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiFormLabel-root` - Root label container
- `.MuiFormLabel-colorPrimary` - Primary color styling
- `.MuiFormLabel-filled` - Applied when the associated input is filled

**State Classes:**

- `.Mui-disabled` - Applied when the label is disabled
- `.Mui-error` - Applied when there's an error state
- `.Mui-focused` - Applied when the associated input is focused
- `.Mui-required` - Applied when the label is for a required field
- `.read-only` - Applied when the readOnly prop is true

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.typography.body1 - Base typography styles for the label
- theme.spacing(0.5) - Margin bottom for the label

**Layout Specifications:**

- Label display: `block`
- White space: `normal` (or `nowrap` if noWrap prop is true)
- Margin bottom: `4px` (theme.spacing(0.5))
- Font size: Inherits from theme.typography.body1
- Line height: Inherits from theme.typography.body1
- Transition: `color 200ms cubic-bezier(0.0, 0, 0.2, 1) 0ms` for smooth state changes
