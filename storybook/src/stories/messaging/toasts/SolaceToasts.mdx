<Subtitle>Component Description</Subtitle>

The Solace Toasts component provides temporary notification messages that appear at the bottom center of the screen to communicate the outcome of user actions or system events. It's useful for providing immediate feedback about operations without interrupting the user's workflow, implementing a toast notification pattern that delivers timely, non-intrusive feedback through brief, automatically dismissing messages.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiSnackbar-root` - Root snackbar container
- `.MuiSnackbar-anchorOriginBottomCenter` - Positioning for bottom center display
- `.alert-toast` - Custom className for toast styling
- `.MuiAlert-root` - Alert component when severity is provided
- `.MuiAlert-icon` - Container for severity icons

**Severity Classes:**

- `.MuiAlert-standardSuccess` - Applied for success toast styling
- `.MuiAlert-standardError` - Applied for error toast styling
- `.MuiAlert-standardInfo` - Applied for info toast styling
- `.MuiAlert-standardWarning` - Applied for warning toast styling

**Custom Classes:**

- Custom padding applied based on severity presence
- Icon mapping with theme-based colors

To customize colors, override the palette values in your theme configuration by extending the base theme and modifying the specific color group and variant you want to change.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing() - Internal padding and margins
- theme.components.MuiSnackbar.styleOverrides.root - Override snackbar container styling
- theme.components.MuiAlert.styleOverrides.root - Override alert styling when severity is used
- theme.typography.body2 - Toast message text styling

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and positioning through snackbar overrides.

**Layout Specifications:**

- Anchor position: Bottom center of viewport
- Auto-hide duration: 4000ms (4 seconds) without action, 8000ms (8 seconds) with action
- Resume hide duration: 1000ms after user interaction
- Padding: 0 16px 0 8px with severity, 0 16px without severity
- Click-away dismissal: Disabled for better user control
- Role: toast for accessibility
