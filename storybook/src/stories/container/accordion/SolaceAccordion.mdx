<Subtitle>Component Description</Subtitle>

The SolaceAccordion component provides an expandable/collapsible container that helps organize content into sections by implementing a progressive disclosure pattern. This approach reduces visual clutter by only revealing essential information up front while keeping users focused on the main task, with secondary or complex content available in expandable areas when needed.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing([NUMBER]) - Padding between elements (e.g., theme.spacing(2) for 16px)
- theme.typography.body1 - Text style for accordion content
- theme.typography.subtitle1 - Text style for accordion headers
- theme.breakpoints.values.[SIZE] - Responsive behavior breakpoints
- theme.components.MuiAccordion.styleOverrides.root - Custom styling for accordion container
- theme.components.MuiAccordionSummary.styleOverrides.root - Custom styling for header section
- theme.components.MuiAccordionDetails.styleOverrides.root - Custom styling for content section

Available mixins for consistent styling:

- theme.mixins.toolbar - Consistent toolbar styling if applicable

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and responsive behavior using breakpoint values.

**Layout Specifications:**

- Summary section padding: `theme.spacing(0.75, 0, 0.75, 1)` (6px 0px 6px 8px)
- Details section padding: `theme.spacing(1, 2, 2, 6)` (8px 16px 16px 48px)
- Expand icon: 24x24px ArrowRight icon
- Transition timeout: `350ms`
