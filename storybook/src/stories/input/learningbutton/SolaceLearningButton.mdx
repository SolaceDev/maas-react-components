<Subtitle>Component Description</Subtitle>

The Solace Learning Button component provides a customizable button for interactive learning experiences, useful for creating engaging call-to-action elements and allowing users to initiate learning processes or navigate through educational content. The component implements a flexible button system with the purpose of enhancing user interaction in learning environments, helping users clearly identify actionable items while maintaining a consistent and appealing visual style across the application.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiButton-root` - Root button container
- `.MuiIconButton-root` - Root icon button container (for icon variant)
- `.call-to-action-button` - Applied to call-to-action variant
- `.dark-call-to-action-button` - Applied to dark call-to-action variant
- `.dark-outline-button` - Applied to dark outline variant

**State Classes:**

- `.Mui-disabled` - Applied when the button is disabled
- `.Mui-focusVisible` - Applied when the button has keyboard focus

**Variant Classes:**

- `.MuiButton-contained` - Applied to call-to-action and dark-call-to-action variants
- `.MuiButton-outlined` - Applied to dark-outline variant
- `.icon-button` - Applied to icon variant

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.typography.button - Typography styles for button text
- theme.spacing(1) - Used for padding and gap calculations
- theme.shape.borderRadius - Border radius for buttons

**Layout Specifications:**

- Button height: `36px` (theme.spacing(4.5))
- Button padding: `8px 16px` (theme.spacing(1, 2))
- Icon button size: `40px` (theme.spacing(5))
- Border radius: Inherits from theme.shape.borderRadius
- Gap between icon and text: `8px` (theme.spacing(1))
- Focus outline offset: `2px`
- Transition: `background-color 250ms cubic-bezier(0.4, 0, 0.2, 1) 0ms, box-shadow 250ms cubic-bezier(0.4, 0, 0.2, 1) 0ms, border-color 250ms cubic-bezier(0.4, 0, 0.2, 1) 0ms, color 250ms cubic-bezier(0.4, 0, 0.2, 1) 0ms`
