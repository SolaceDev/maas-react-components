<Subtitle>Component Description</Subtitle>

The Solace Text Area component is a versatile and customizable multi-line text input field designed for use in forms and other UI elements, providing a consistent way to collect longer text input from users while offering various styling options and states. This component extends the functionality of a standard HTML textarea, adding features such as label support, error handling, helper text, and customizable styling, making it particularly useful for creating accessible and visually consistent form layouts that require longer text input.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiTextField-root` - Root text area container
- `.MuiInputBase-root` - Input base container
- `.MuiInputBase-input` - Input element
- `.MuiFormLabel-root` - Label element
- `.MuiFormHelperText-root` - Helper text container

**State Classes:**

- `.Mui-focused` - Applied when the text area has focus
- `.Mui-disabled` - Applied when the text area is disabled
- `.Mui-error` - Applied when hasErrors is true
- `.Mui-required` - Applied when the field is required

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(4) - Height of the input field
- theme.typography.body1 - Font styling for input text
- theme.shape.borderRadius - Border radius of the input field
- theme.components.MuiTextField.styleOverrides.root - Customize the root text area container
- theme.components.MuiInputBase.styleOverrides.root - Customize the input base container
- theme.components.MuiInputBase.styleOverrides.input - Customize the input element

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and shape using the borderRadius property.

**Layout Specifications:**

- Minimum rows: Controlled by the minRows prop (default: 4)
- Maximum rows: Controlled by the maxRows prop (default: 4)
- Width: Can be controlled by the width prop or fullWidth prop
- Resizability: Controlled by the resizable prop
- Padding: 16.5px 14px (theme.spacing(2, 1.75))
- Font size: 1rem (16px)
- Line height: 1.4375em (23px)
