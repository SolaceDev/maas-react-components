<Subtitle>Component Description</Subtitle>

The Solace List component provides a simple and flexible way to display collections of items in a vertical list layout. It's a lightweight wrapper around Material-UI's List component that follows Solace design guidelines while maintaining full compatibility with Material-UI's list ecosystem.

The component implements a standard list layout pattern with the purpose of organizing content into a scannable, vertical structure. This pattern is ideal for displaying collections of related items, navigation menus, or any content that benefits from a simple, linear arrangement.

<Subtitle>When To Use It</Subtitle>

Use Solace List when you need to display collections of items in a simple, vertical layout without complex interactions or data management features. This basic list pattern is particularly effective for straightforward content presentation.

Common use cases include:

- Simple navigation menus with list items and buttons
- Basic content listings without selection or complex interactions
- Static information displays where items are presented sequentially
- Foundation for more complex list components when combined with SolaceListItem and SolaceListItemButton

<Subtitle>When Not To Use It</Subtitle>

- For data with multi-selection capabilities - use SolaceGridListMultiSelect for checkbox-based selection
- When you need structured grid layouts - use SolaceGridList for tabular data presentation
- For complex data tables with sorting and filtering - use SolaceTable for full table functionality
- When you need advanced list features like virtualization - use SolaceGridList with virtualization options
- For card-based layouts - use SolaceCard components for richer content presentation
- When displaying hierarchical data - consider SolaceTree for parent-child relationships

<Subtitle>CSS Classes / Styles</Subtitle>

**Material-UI List Classes:**

Since SolaceList is a wrapper around Material-UI's List component, it inherits all the standard Material-UI list styling classes:

- `.MuiList-root` - Root list container
- `.MuiList-padding` - Applied when disablePadding is false (default)
- `.MuiList-dense` - Applied when dense prop is true for compact spacing
- `.MuiList-subheader` - Applied to subheader elements

**List Item Classes (when using SolaceListItem):**

- `.MuiListItem-root` - Root list item container
- `.MuiListItem-dense` - Applied for compact list item spacing
- `.MuiListItem-alignItemsFlexStart` - When alignItems="flex-start"
- `.MuiListItem-divider` - Applied when divider prop is true
- `.MuiListItem-gutters` - Default padding (removed when disableGutters=true)
- `.MuiListItem-padding` - Applied when disablePadding is false

**List Item Button Classes (when using SolaceListItemButton):**

- `.MuiListItemButton-root` - Root list item button container
- `.MuiListItemButton-dense` - Applied for compact button spacing
- `.MuiListItemButton-alignItemsFlexStart` - When alignItems="flex-start"
- `.MuiListItemButton-divider` - Applied when divider prop is true
- `.MuiListItemButton-gutters` - Default padding (removed when disableGutters=true)
- `.Mui-selected` - Applied when selected prop is true
- `.Mui-disabled` - Applied when disabled prop is true
- `.Mui-focusVisible` - Applied during keyboard focus

**Theme Customization:**
The component uses Material-UI's theme system. Key theme paths for customization:

- `theme.components.MuiList` - Override list component styles
- `theme.components.MuiListItem` - Override list item styles
- `theme.components.MuiListItemButton` - Override list item button styles
- `theme.spacing()` - Control padding and margins
- `theme.palette` - Control colors and text styling

**Layout Specifications:**

- Default vertical layout with flex column direction
- Automatic spacing between items based on Material-UI defaults
- Dense mode reduces vertical padding for more compact display
- Subheader support for section grouping
- Full width container by default with optional custom styling via sx prop
