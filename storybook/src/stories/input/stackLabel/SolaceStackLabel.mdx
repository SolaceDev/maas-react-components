<Subtitle>Component Description</Subtitle>

The Solace Stack Label component is a customizable label element designed for use in forms and other UI elements. It provides a consistent way to label form inputs and other components while offering various styling options and states.

This component extends the functionality of a standard HTML label, adding features such as required indicators, disabled states, and customizable text styles. It's particularly useful for creating accessible and visually consistent form layouts.

<Subtitle>When To Use It</Subtitle>

Use Solace Stack Label in the following scenarios:

- Labeling form inputs (text fields, select dropdowns, checkboxes, etc.)
- Creating section headers in forms or other UI components
- Providing descriptive text for UI elements that require explanation
- Implementing accessible form designs that clearly associate labels with their corresponding inputs
- When you need a label with customizable styles (e.g., bold, large text) to fit your UI design

<Subtitle>When Not To Use It</Subtitle>

- For general text content that doesn't act as a label (use standard text components instead)
- In situations where inline labels are more appropriate (consider using SolaceLabel for inline scenarios)
- For complex label requirements that go beyond text and basic styling (consider creating a custom component)

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiFormLabel-root` - Root label container
- `.MuiFormLabel-asterisk` - Required indicator asterisk

**State Classes:**

- `.Mui-disabled` - Applied when the label is disabled
- `.Mui-required` - Applied when the label is marked as required
- `.bold-label` - Applied when the bold prop is true
- `.read-only` - Applied when the readOnly prop is true

**Theme Customization:**
The component uses Material-UI's theme system with custom Solace styling. Key theme paths:

- `theme.typography.body1` - Default font size and style
- `theme.typography.subtitle1` - Font size and style when large prop is true
- `theme.palette.text.primary` - Text color
- `theme.palette.text.disabled` - Disabled text color

**Layout Specifications:**

- Display: block (stacked layout)
- Font size: Controlled by the large prop (body1 or subtitle1)
- Font weight: Controlled by the bold prop
- White space: Controlled by the noWrap prop
