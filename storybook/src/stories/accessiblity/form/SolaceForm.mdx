<Subtitle>Component Description</Subtitle>

A Solace Form is any page where users can input data that will be either saved or applied to be saved later, either automatically or manually. Forms can be presented as full-page layouts or within action dialogs, with all forms containing three essential elements: a header with title, at least one input component, and a footer with action buttons.

<Subtitle>Usage</Subtitle>

### What do I use when?

There are 2 main areas to place Forms:

- Within the main portion of the screen
- Inside a Dialog

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Layout Classes:**

- `.CreateContentLayout` - Root form container with full height and overflow handling
- `.FormContainer` - Two-column grid layout (108px label column, flexible content column)
- `.SectionTitle` - Section headers with consistent typography and spacing
- `.LabelColumn` - Left column for form labels with consistent padding
- `.ValueColumn` - Right column for form controls with responsive width options

**Grid Layout Classes:**

- `.fixed-width-520` - Fixed width container for standard form controls
- `.fixed-width-260` - Narrower width container for compact controls
- `.TwoColumnSpan` - Spans both grid columns for full-width content

**Interactive Element Classes:**

- `.TabsContainer` - Horizontal flex container for tabs with border styling
- `.TabLayoutContainer` - Content area below tabs with proper spacing
- `.FilterWrapper` - Horizontal layout for filter controls with consistent gaps
- `.IconContainer` - Flex container for icons with proper alignment
- `.SelectorIconContainer` - Container for selector icons with end alignment

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing([NUMBER]) - Controls consistent spacing throughout the form:
  - Grid column gap: theme.spacing(36/8) (18px)
  - Grid row gap: theme.spacing(2) (16px)
  - Form container padding: theme.spacing(2) 0 theme.spacing(1) 0
  - Section title margin: theme.spacing(4) top margin
  - Filter wrapper gap: theme.spacing(3) (24px)
- theme.typography.body1 - Text style for form labels and content
- theme.typography.h6 - Text style for section titles
- theme.components.MuiFormControl.styleOverrides - Custom styling for form control elements
- theme.components.MuiFormLabel.styleOverrides - Custom styling for form labels

**Layout Specifications:**

- Form container: Two-column grid layout (108px label column, flexible content column)
- Label column: 4px top padding for alignment with inputs
- Content column: Responsive width with options for 520px or 260px fixed widths
- Section spacing: Consistent 32px (theme.spacing(4)) margins between sections
- Tab container: Full width with 1px borders and consistent padding
