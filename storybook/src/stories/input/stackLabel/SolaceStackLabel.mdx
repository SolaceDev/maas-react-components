<Subtitle>Component Description</Subtitle>

The Solace Stack Label component is a customizable label element designed for use in forms and other UI elements, providing a consistent way to label form inputs and other components while offering various styling options and states. This component extends the functionality of a standard HTML label, adding features such as required indicators, disabled states, and customizable text styles, making it particularly useful for creating accessible and visually consistent form layouts.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.typography.body1 - Default font size and style
- theme.typography.subtitle1 - Font size and style when large prop is true
- theme.components.MuiFormLabel.styleOverrides.root - Customize the root label container
- theme.components.MuiFormLabel.styleOverrides.asterisk - Customize the required indicator asterisk

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify typography using predefined variants.

**Layout Specifications:**

- Display: block (stacked layout)
- Font size: Controlled by the large prop (body1 or subtitle1)
- Font weight: Controlled by the bold prop (400 for normal, 700 for bold)
- White space: Controlled by the noWrap prop (normal or nowrap)
- Margin bottom: 8px (theme.spacing(1))
