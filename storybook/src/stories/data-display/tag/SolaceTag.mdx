<Subtitle>Component Description</Subtitle>

The SolaceTag component provides a visual label element for categorizing, organizing, or highlighting content with customizable styling options. It's useful for content classification, metadata display, or visual organization systems, allowing users to quickly identify and group related information through consistent visual markers.

The component implements a flexible tagging pattern with the purpose of providing visual categorization without complex interactions. This pattern helps organize information hierarchically while offering extensive customization options for colors, variants, and visual styling to match different content types and organizational schemes.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.typography.body2 - Base typography for tag text
- theme.shape.borderRadius - Border radius for tag container
- theme.transitions.create(['background-color', 'box-shadow']) - Tag hover and focus transitions
- theme.components.MuiChip.styleOverrides.root - Custom tag styling
- theme.components.MuiChip.styleOverrides.label - Custom label styling

Available mixins for consistent styling:

- theme.mixins.tag.filled - Filled variant styling
- theme.mixins.tag.outlined - Outlined variant styling

**Layout Specifications:**

- Height: Fixed at `18px` (sm size from CHIP_PX_HEIGHTS)
- Border radius: Fixed at `20px` (xs size from CHIP_PX_BORDER_RADIUS)
- Font size: Fixed at `12px` (xs size from BASE_FONT_PX_SIZES)
- Font weight: `400` (regular)
- Default variant: `FILLED`
- Default max width: `100%`
- Padding: `2px` left and right
- Icon size: `14x14` pixels for leading icons
- Size: Always `medium` (not responsive to content)
- Clickable hover effect: Slight opacity change and elevation increase
