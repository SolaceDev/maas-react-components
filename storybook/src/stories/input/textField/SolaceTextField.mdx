<Subtitle>Component Description</Subtitle>

The Solace Text Field component is a versatile and customizable single-line text input field designed for use in forms and other UI elements. It provides a consistent way to collect user input while offering various styling options, validation states, and customization features.

This component extends the functionality of a standard HTML input field, adding features such as label support, error/warning states, helper text, custom icons, and adornments. It's particularly useful for creating accessible and visually consistent form layouts that require single-line text input.

<Subtitle>When To Use It</Subtitle>

Use Solace Text Field in the following scenarios:

- Creating form inputs for single-line text entry
- Collecting user input like names, emails, passwords, or URLs
- Implementing search fields with custom icons
- When you need input validation with error/warning states
- Creating fields with custom adornments (icons, buttons, etc.)
- When you need to provide helper text or validation messages
- For fields that require specific input types (text, number, email, etc.)
- When you need controlled input with state management

<Subtitle>When Not To Use It</Subtitle>

- For multi-line text input (use SolaceTextArea instead)
- When you need rich text editing capabilities
- For complex input patterns that require custom formatting
- When you need a more specialized input type (use appropriate components like SolaceDatePicker)
- If you need to display read-only text without any input functionality
- When the input requires a specific selection from predefined options (use SolaceSelect instead)

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiTextField-root` - Root text field container
- `.MuiInputBase-root` - Input base container
- `.MuiInputBase-input` - Input element
- `.MuiFormLabel-root` - Label element
- `.MuiFormHelperText-root` - Helper text container
- `.MuiInputAdornment-root` - Container for start/end adornments

**State Classes:**

- `.Mui-focused` - Applied when the text field has focus
- `.Mui-disabled` - Applied when the text field is disabled
- `.Mui-error` - Applied when hasErrors is true
- `.Mui-required` - Applied when the field is required
- `.read-only` - Applied when readOnly is true

**Theme Customization:**
The component uses Material-UI's theme system with custom Solace styling. Key theme paths:

- `theme.spacing(4)` - Height of the input field
- `theme.palette.text.primary` - Text color
- `theme.palette.error.main` - Error state color
- `theme.palette.warning.main` - Warning state color
- `theme.typography.body1` - Font styling for input text

**Layout Specifications:**

- Height: `theme.spacing(4)`
- Width: Controlled by width prop or fullWidth prop
- Minimum width: Controlled by minWidth prop
- Input size: Controlled by size prop (character width)
- Icon size: Standard Material-UI icon size
