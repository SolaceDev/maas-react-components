<Subtitle>Component Description</Subtitle>

The Solace Checkbox component provides a customizable checkbox input for user selection, useful for allowing users to make binary choices, enabling or disabling options, and creating lists of selectable items. The component implements a form input pattern with the purpose of capturing user preferences or acknowledgments, helping users make clear decisions while providing visual feedback and supporting various states such as checked, unchecked, and indeterminate.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiCheckbox-root` - Root checkbox container
- `.MuiCheckbox-checked` - Applied when the checkbox is checked
- `.MuiCheckbox-indeterminate` - Applied when the checkbox is in an indeterminate state
- `.MuiFormControlLabel-root` - Container for the checkbox and its label

**State Classes:**

- `.Mui-checked` - Applied when the checkbox is checked
- `.Mui-disabled` - Applied when the checkbox is disabled
- `.Mui-error` - Applied when the checkbox has an error state
- `.Mui-focused` - Applied when the checkbox has focus

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(1) - Padding between checkbox and label
- theme.typography.body1 - Text style for checkbox label
- theme.components.MuiCheckbox.styleOverrides.root - Base checkbox styling
- theme.components.MuiFormControlLabel.styleOverrides.root - Label container styling

Available mixins for consistent styling:

- theme.mixins.checkbox.base - Standard checkbox styling patterns
