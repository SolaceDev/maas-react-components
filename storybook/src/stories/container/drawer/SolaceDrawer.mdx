<Subtitle>Component Description</Subtitle>

The SolaceDrawer component provides a sliding panel that enters from the edge of the screen to display additional content, navigation options, or contextual information without navigating away from the current view. This persistent drawer pattern creates a non-modal overlay that can coexist with the main content, providing additional workspace without completely interrupting the user's current task while offering optional resizing capabilities for user customization.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing([NUMBER]) - Padding and margin (e.g., theme.spacing(1) for 8px content margin)
- theme.typography.[VARIANT] - Text styles for drawer content
- theme.breakpoints.values.[SIZE] - Responsive behavior breakpoints
- theme.components.MuiDrawer.styleOverrides.root - Custom styling for drawer container
- theme.components.MuiDrawer.styleOverrides.paper - Custom styling for drawer paper/content area

Available mixins for consistent styling:

- theme.mixins.toolbar - Consistent toolbar styling if applicable in drawer content

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and responsive behavior using breakpoint values.

**Layout Specifications:**

- Default width: `320px`
- Minimum width: `100px` (when resizable)
- Maximum width: `1000px` (when resizable)
- Resize handle width: `8px`
- Border radius: `2px`
- Content margin: `theme.spacing(1)` (8px, left or right based on anchor)
- Box shadow: `0px 2px 4px` with theme color
- Variant: `persistent` (non-modal)
- Transition timeout: `225ms`
