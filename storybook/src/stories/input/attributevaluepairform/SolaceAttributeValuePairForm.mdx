<Subtitle>Component Description</Subtitle>

The Solace Attribute Value Pair Form component provides a dynamic form for managing key-value pairs, useful for creating and editing lists of attributes and their corresponding values and allowing users to add, remove, and reorder pairs as needed. The component implements a flexible input pattern with the purpose of capturing structured data in a key-value format, helping users input and manage sets of related information while providing features like validation, reordering, and bulk editing.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(2) - Padding between form elements
- theme.typography.body1 - Text style for input fields
- theme.typography.caption - Text style for helper text
- theme.components.MuiFormControl.styleOverrides.root - Base form control styling
- theme.components.MuiInputBase.styleOverrides.root - Input field container styling

Available mixins for consistent styling:

- theme.mixins.gutters - Consistent padding for form container

**Layout Specifications:**

- Form container padding: `theme.spacing(2)`
- Input field margins: `theme.spacing(1) 0`
- Reorder button size: `36px` height
- Add/Remove button placement: `theme.spacing(2)` from last input pair
- Transition timeout: `300ms`
