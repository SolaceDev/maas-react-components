<Subtitle>Component Description</Subtitle>

The Solace Color Picker component provides a grid-based selection interface for choosing colors from predefined color palettes, useful for applications where users need to select visual identifiers like environment colors, status indicators, or category colors and allowing users to make selections from an organized grid layout with visual color previews. The component implements a custom dropdown select pattern with the purpose of making color selection intuitive through visual grid layouts, helping users quickly identify and select colors through visual representation while maintaining form consistency and providing accessible color selection workflows.

<Subtitle>CSS Classes / Styles</Subtitle>

**Main Component Classes:**

- `.MuiSelect-root` - Root select container for the color picker dropdown
- `.SolacePickerGrid` - Custom grid layout class applied to the MenuList for organizing color options
- `.MuiMenuItem-root` - Individual color option styling within the dropdown menu
- `.MuiOutlinedInput-root` - Input field container styling with outline border

**State Classes:**

- `.Mui-disabled` - Applied when color picker is disabled, graying out the component
- `.Mui-error` - Applied when picker has validation errors, showing red error styling
- `.Mui-focused` - Applied when picker receives focus, highlighting the input border
- `.Mui-selected` - Applied to currently selected color menu item

**Custom Classes:**

- `SolacePickerIconWrapper` - Icon/color container with centered alignment and fixed dimensions
- `Color` - Colored square display component with border, typography, and "Aa" preview text
- `EmptyMessageContainer` - Centered empty state message styling for when no colors are available

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(3) - Color item size (24px width and height)
- theme.spacing(1) - Gap between grid color items (8px)
- theme.typography.caption - Font style for "Aa" preview text
- theme.components.MuiSelect.styleOverrides.root - Select input container customization
- theme.components.MuiMenuItem.styleOverrides.root - Color option item customization

Available mixins for consistent styling:

- theme.mixins.toolbar - Used for maintaining consistent height with toolbar components

**Layout Specifications:**

- Color item dimensions: theme.spacing(3) × theme.spacing(3) (24px × 24px)
- Grid gap: theme.spacing(1) (8px) between items
- Grid layout: 5 columns with equal distribution
- Maximum rows: 5 before scrolling
- Dropdown content padding: theme.spacing(2) (16px)
- Select input padding: 4px 40px 4px 4px
- Color preview border radius: theme.spacing(0.5) (4px)
- Menu max height: calc(100% - 96px)
