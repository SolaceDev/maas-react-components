<Subtitle>Component Description</Subtitle>

The SolaceGridListMultiSelect component provides a grid layout with multi-selection capabilities for displaying collections of selectable items, useful for organizing content into structured grids where users need to select multiple items and enabling efficient browsing and batch selection while maintaining consistent visual alignment and clear selection states. This selectable grid pattern enables users to choose multiple items from a collection in an organized, scannable format, helping users efficiently make bulk selections while providing clear visual feedback about selected states and maintaining flexible layout options.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing(2) - Default padding between grid items
- theme.typography.body1 - Base typography for grid content
- theme.components.MuiCheckbox.styleOverrides.root - Custom checkbox styling
- theme.components.MuiImageList.styleOverrides.root - Custom grid container styling
- theme.transitions.create(['background-color']) - Selection state transitions

Available mixins for consistent styling:

- theme.mixins.gridList.row - Standard row styling
- theme.mixins.gridList.selected - Selected state styling
- theme.mixins.gridList.checkbox - Checkbox container styling

**Layout Specifications:**

- Grid template columns: Defined by the `gridTemplate` prop using CSS Grid syntax
- Item height: Calculated dynamically using `getGridListItemHeight()` utility
- Virtual list height: Configurable through `virtualizedListOption.height`
- Checkbox column width: Fixed at `48px`
- Row padding: `8px` (theme.spacing(1))
- Selection transition: `200ms` background-color fade
- Hover state opacity: 0.04
- Selected state opacity: 0.08
- Checkbox position: Left-aligned with `16px` margin
