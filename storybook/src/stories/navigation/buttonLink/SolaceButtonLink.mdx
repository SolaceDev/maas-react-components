`SolaceButton` (variant="link")

<Subtitle>Component Description</Subtitle>

The Solace Button Link component provides clickable text elements that function as navigation links or interactive buttons. It's useful for creating inline links within text content, allowing users to trigger actions or navigate to other locations without the visual weight of a traditional button.

The component implements a button link pattern with the purpose of providing consistent navigation actions that maintain button styling while offering link functionality. This pattern helps create cohesive interfaces where navigation elements need to blend seamlessly with other interactive components.

**Non-Color Theming:**
Structural and layout customization through component overrides and mixins:

- theme.spacing() - Padding and margin customization through sx prop
- theme.components.MuiLink.styleOverrides.root - Override link styling
- theme.typography.body2 - Link text typography styling
- theme.shape.borderRadius - Border radius for standard variant

To customize layout and structure, override the component styleOverrides in your theme configuration. You can modify spacing using the theme spacing function, typography using predefined variants, and link behavior through link overrides.

**Layout Specifications:**

- Dense variant: padding: 0 for inline use
- Standard variant: padding: 6px 16px with minWidth: 100px and height: 32px
- Focus outline: 1px solid accent color with outlineOffset: 1px
- Border radius: 4px for standard variant
- External link icon spacing: marginRight: 4px
